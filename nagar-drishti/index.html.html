From pothole tracking to predictive infrastructure alerts — Nagar-Drishti sees what humans miss.

📡
Real-time Issue Tracking

Citizens report civic issues via app or SMS. AI auto-classifies, prioritizes, and routes to the right department within seconds.

🧠
Gemini AI Analysis

Natural language processing on complaint data. Extracts patterns, sentiment, and urgency signals using Gemini API for deep contextual understanding.

⚡
n8n Workflow Automation

Automated escalation workflows via n8n. Critical issues auto-trigger alerts, assign officials, and send status updates without manual intervention.

How Nagar-Drishti works

Three intelligent layers working together to transform civic complaint data into actionable governance.

01
Citizen Reports Issue

Via web app, mobile, or SMS — citizens submit civic complaints with photos and location. React frontend captures it instantly.

→
02
Gemini AI Processes

FastAPI backend sends data to Gemini API. AI classifies issue type, severity, and affected zone. n8n triggers automated routing workflows.

→
03
Officials Act Fast

Department gets instant alert with AI-generated priority score. Dashboard tracks resolution. Citizens get SMS status updates automatically.
